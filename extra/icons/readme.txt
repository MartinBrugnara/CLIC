Given clic.png (1024x1024)

* assets/icons/win/clic.ico
    Convert it with https://convertico.com/ to ico for windows

* assets/icons/mac/clic.icns
    Invoke png2icns.sh, it will produce the file. Then move it.

* assets/icons/png/*.png
    Invoke png2icns.sh, it will produce clic.iconset, rename it in icons/png.
    Icons will be called icon_<resolution>.png, icon_ prefix.
    Finally add clic.png as png/1024x1024.png
